Learning HTML

Learning HTML is easy, but it takes a bit of time and effort.

First, we should find out, what is HTML.
Here is a quote from Wikipedia:

Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

Some elements I have used to practice html

There are many tags in html that have a semantic meaning and there are some that don't have it and instead of those it is better to use other tags or even CSS.

So for example, it is not advised to use the tag <small>, but instead - to use CSS for setting the size of a font. But here I am gonna use it anyway to make the word 'small' appear small.

There is a tag that was used to make text in italics, but is now sometimes used for icons - <i>. Now with HTML5 it is advised to use the tag <em> for emphasizing text, for example - I like tea, but I love coffee.

Attention! Now some other tags that do have a semantic meaning. I just used - <strong> to say that something in the text is important, in this case just one word, but it can be a phrase.
